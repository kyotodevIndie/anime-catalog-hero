### Installation

1. Clone the repo

```sh
git clone https://github.com/kyotodevIndie/anime-catalog-hero.git
```

3. Install the packages using NPM or Yarn

```sh
yarn install
```

```sh
npm install
```
4. run the project

```sh
yarn dev
```

```sh
npm dev
```

5. Happy coding!
